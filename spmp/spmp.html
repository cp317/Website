Introduction

Study Space is a web-based application that allows students in any university to post and find study group locations. The purpose of this application is to allow students to find study groups with ease. By doing so, students help each other achieve their academic goals as well as make new acquaintances.

Purpose

This document outlines the project's deliverable, milestones, organization, management, and technical details.

Project Organization

Organizational Structure

For each task a group is assigned to complete it. Within these groups, there are managers that oversee the task’s progress. These managers also communicate with each other should their tasks be related. The groups are responsible for making the decisions for their tasks.

Managerial Process

Management Objectives and Priorities

The primary goal of this project is to create a web application that people use to find study locations for certain courses. This project should be completed before August 9, 2017. The budget is no concern, as there is no budget to go over.

Assumptions, Dependencies, and Constraints

The assumption being made for this project is that the application is being used in an environment where internet is readily available. This project is dependent on the knowledge of members, as certain technologies may require members to learn how to use it. The constraints for this project is the time available and the limited funds.

Monitoring and Controlling Mechanisms

This project will be monitored and controlled by having teams perform software quality assurance on all section of the project, having a shared GitHub repository, and a manager to keep groups on schedule.

Technical Process

Methods, Tools, and Techniques

For this project, all documents are stored on GitHub under the repository “cp317”. This repository stores software quality assurance remarks under the issues tab. The application uses TypeScript, HTML, and CSS. In addition, uses these Angularjs, Bootstap, Firebase, Google Maps API, MaterializeCSS, Nodejs, and TS-MD5.
